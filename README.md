| 카테고리 헤더 | 텍스트만 (작게) | "종합몰 ─────────" (PC 사이드패널 톤) |

## 검증 포인트

- 카테고리명 옆에 가로 구분선이 화면 끝까지 늘어나는지
- 아이콘이 셀 안에서 70% 정도 차지하면서 여백 충분한지
- 미니멀 톤이 전반적으로 살아났는지

## 다음 미세조정 가능 (사용자 보고 결정)

- 70%가 너무 작으면 → 75% 또는 80%로 조정
- 카테고리 헤더가 너무 약하면 → font-weight 또는 색상 강화
- 격자 gap이 어색하면 → gap-2 → gap-3 등 조정
